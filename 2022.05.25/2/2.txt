С 20 версии C++ можно использовать erase_if, которая правильно стирает элементы,
а не как remove_if. Remove_if - все те элементы, которые нужно не нужно удалять - 
перемещает их в начало диапазона, сохраняя относительный порядок элементов.
Это происходит за один проход через диапазон данных. Потом он возвращает итератор, 
который указывает на первый элемент из получившегося хвоста ненужных элементов, 
чтобы можно было их удалить с помощью erase.
Единственное, remove_if не работает для контейнеров типо set (const_iterator).
Erase_if работает с vector, list и set.
С 17 версии в remove_if есть ExecutionPolicy - это С++ класс, используемый в 
качестве уникального типа для перегрузки алгоритмов STL.
А erase_if только с 20 версии, поэтому различий не может быть в принципе.
